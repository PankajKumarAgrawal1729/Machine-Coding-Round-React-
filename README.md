# Project Summary

## Overview

This project consists of three tasks integrated within a single React application. The tasks include an OTP Form, a Drag and Drop Course Cards List, and a Paginated Data Table with Search functionality. Each task is accessible via different URL routes, ensuring a seamless navigation experience.

## Tasks

### 1. OTP Form

**Route:** `/otp-form`

**States:**
- **Empty Form State:** A form to input a 4-digit OTP.
- **Filling State:** The state during the input of the OTP.
- **Filled State (Success):** The state when the OTP is correctly filled.
- **Filled State (Error):** The state when the OTP is incorrectly filled.

### 2. Drag and Drop Course Cards

**Route:** `/course-list`

**Features:**
- A list of vertical course cards implemented using a drag-and-drop library.
- Each card includes actions for "Move to Top," "Move to Bottom," and "Remove."

### 3. Data Table

**Route:** `/batches`

**Features:**
- A paginated table listing batches of a course and their statuses.
- A search option to filter the batches.

## Additional Requirements

- The base URL (`/`) redirects to the OTP form route (`/otp-form`).
- The branding logo is positioned in the bottom right corner for all tasks.
- Clicking the branding logo opens the "chaicode.com" website.

## Project Setup

1. **Clone the repository:**
   ```sh
   git clone https://github.com/PankajKumarAgrawal1729/Machine-Coding-Round-React-.git
   cd Machine-Coding-Round-React-
   ```

2. **Install dependencies:**
   ```sh
   npm install
   ```

3. **Start the application:**
   ```sh
   npm start
   ```

## Navigation

- **OTP Form:** [http://localhost:3000/otp-form](http://localhost:3000/otp-form)
- **Course List:** [http://localhost:3000/course-list](http://localhost:3000/course-list)
- **Batches:** [http://localhost:3000/batches](http://localhost:3000/batches)

## Branding Logo

Ensure the branding logo is placed at the bottom right corner of each task's page. Clicking the logo should navigate to [chaicode.com](https://chaicode.com).

## Conclusion

This project demonstrates the implementation of various React features and functionalities, providing a comprehensive solution for form handling, drag-and-drop interactions, and data table management.
